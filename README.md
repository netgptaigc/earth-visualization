# 实时活动地球可视化项目
🌍 核心功能

• 基础地球可视化: 高分辨率地球纹理，包含地形、海洋和云层
• 动态效果: 地球自转、云层独立运动
• 真实感渲染: 法线贴图、高光贴图、大气层效果
• 星空背景: 15000颗动态星星营造宇宙环境
🛰️ 高级功能

• 卫星系统: 8颗动画卫星在不同轨道运行
• 轨道可视化: 多层卫星轨道环
• 交互控制: 鼠标拖拽旋转、滚轮缩放
• 实时控制面板: 调节自转速度、云层速度、光照强度
📊 数据驱动版本

• 全球数据点: 随机分布的数据点模拟全球活动
• 事件可视化: 活跃数据点以绿色高亮显示
• 性能监控: 实时FPS、对象计数、旋转角度等统计
• 动态激活: 随机激活数据点模拟实时事件流
🚀 技术特点

• 使用 Three.js 进行 WebGL 渲染
• 自定义着色器实现大气层效果
• 实时光照系统模拟太阳光
• 性能优化确保流畅动画
📁 项目文件

• index.html: 基础版实时地球
• advanced-earth.html: 增强版，包含更多交互功能
• data-driven-earth.html: 数据驱动版本，全球数据可视化
• server.js: 本地开发服务器
• README.md: 详细使用说明
• DEPLOYMENT.md: 部署指南
• package.json: 项目配置和依赖
• start.sh: 快速启动脚本
此目录包含可以直接部署到 GitHub Pages 的所有文件。
